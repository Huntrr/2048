Note: This site is a derivative of the original 2048, by Gabriele Cirulli (which you can find here). This site serves as a testing ground for various algorithms meant to 'beat' the game by optimizing the chances of success. All mechanics have been kept the same, with the only changes to the original code being several interfaces to allow other scripts to see the state of the game and send input to the game.
